# The Brain Song — Affiliate Claim Compliance Rules

Apply to every page, section, heading, bullet, testimonial, FAQ, meta tag, and schema description.

## BANNED WORDS & PHRASES (never use)

### Memory Loss Claims
- cure memory loss
- prevent memory loss
- stop memory loss
- fixes memory loss

### Serious Conditions (never mention)
- dementia
- Alzheimer's
- cognitive decline
- stop or prevent cognitive decline
- treats cognitive decline or memory conditions

### Reversal Claims
- reverse memory loss
- restore memory
- revive memory loss
- rebuild memory
- regrow memory cells
- reverse brain aging
- any language implying people will regain or reverse memory loss

### Misleading Implications
- Any absolute promise ("this will give you better memory")
- Any guarantee of specific results
- Any language implying medical diagnosis, treatment, cure, or prevention

---

## COMPLIANT TERMS (use these)

### Descriptive Terms
- aging brain
- senior moments
- brain fog
- fuzzy thinking
- brain power

### Soft Benefit Statements (always use "supports", "may", "linked to", "promotes")
- better memory
- feeling alert
- feeling mentally sharp
- sharper memory
- clearer thinking
- lift in mental energy

## WRITING RULES

1. **Use soft qualifiers always**: "supports", "may", "linked to", "promotes", "helps", "is associated with"
2. **Net impression matters**: The overall message must not imply a medical cure even if individual words are compliant
3. **No absolute promises**: Never say "will", "guarantees", "eliminates", "completely"
4. **Brainwave claims format**: "Gamma brainwave activity is linked to sharper thinking" — not "Gamma fixes your memory"
5. **BDNF claims format**: "BDNF supports healthy brain function" — not "BDNF reverses memory loss"
6. **Focus on wellness, not treatment**: The product is a wellness audio tool, not medicine
